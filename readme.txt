Welcome to Django of our Project.

First make a venv Virtuell Environment:

python -m venv venv

and activate the venv:

source venv/bin/activate (Linux , Mac)

Windows:
# In cmd.exe
venv\Scripts\activate.bat
# In PowerShell
venv\Scripts\Activate.ps1

than the requirements of Django:
 
pip install -r requirements.txt

hint:
pip list (Show you all the requirements after you install it)

to run the Django Server type:

manage.py runserver
